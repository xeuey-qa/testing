# Пример баг-репорта

**Заголовок:** Аномальная температура (+500°C) в приложении "Погода"
**ID:** BUG-001
**Severity:** Major (Критическая)
**Priority:** High (Высокий)

### Шаги воспроизведения:
1. Открыть сайт/приложение "Погода".
2. В строке поиска ввести "Екатеринбург".
3. Нажать Enter.

### Фактический результат:
На главном экране отображается температура `+500°C`.
*Примечание:* В DevTools (Network) запрос к API возвращает JSON: `{"temp": 500}`.

### Ожидаемый результат:
Отображение корректной текущей температуры (например, +15°C) или ошибка валидации данных на стороне бэкенда.
