# Чек-лист: Тестирование поисковой строки Яндекса

## Позитивные проверки (Positive)
- [ ] Поиск по точному слову (например, "купить велосипед")
- [ ] Поиск по артикулу товара или серийному номеру
- [ ] Работа подсказок (саджестов) при вводе первых букв
- [ ] Переход на вторую страницу выдачи

## Негативные проверки (Negative)
- [ ] Ввод очень длинного запроса (более 1000 символов)
- [ ] Поиск по пустому запросу (нажать "Найти" с пустой строкой)
- [ ] Ввод спецсимволов и скриптов (например, `<script>alert(1)</script>`)

## Проверки удобства (UI/UX)
- [ ] Кнопка очистки строки поиска (крестик)
- [ ] Корректное отображение на мобильном телефоне
